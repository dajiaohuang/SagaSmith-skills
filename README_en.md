# SagaSmith D&D Skills

Portable D&D 5e 2014/2024 Agent Skills.

Runtime mode requires `sagasmith-core` and `sagasmith-dnd`. Every supported
agent platform, including nanobot, operates the same compact
`sagasmith-dnd --json` CLI. This repository contains no database, vector
runtime, game engine, or platform-specific tools.

Without the Runtime, the bundled SRD files remain available for static,
non-persistent guidance.
